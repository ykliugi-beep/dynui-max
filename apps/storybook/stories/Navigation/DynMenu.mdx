# DynMenu

Dropdown context menu component with keyboard navigation and submenus.

## Features

- **Keyboard Navigation**: Arrow keys, Enter, Escape support
- **Nested Menus**: Support for submenus and hierarchical content
- **Accessibility**: Full WCAG 2.1 AA compliance
- **Positioning**: Smart positioning to stay within viewport
- **Custom Triggers**: Flexible trigger mechanisms
- **Dividers**: Visual separation for menu sections

## Usage Examples

- Context menus
- Dropdown navigation
- Action menus
- Settings panels

## Best Practices

- Group related actions together
- Use dividers for logical sections
- Provide keyboard shortcuts when applicable
- Test positioning on different screen sizes
