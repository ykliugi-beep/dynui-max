# DynBreadcrumb

Navigation breadcrumbs component with overflow handling and accessibility features.

## Features

- **Hierarchical Navigation**: Clear path indication
- **Overflow Handling**: Ellipsis for long breadcrumb chains
- **Keyboard Navigation**: Full keyboard support
- **Accessibility**: Proper ARIA labels and landmarks
- **Responsive Design**: Adapts to different screen sizes
- **Custom Separators**: Configurable separator icons

## Usage Examples

- File system navigation
- E-commerce category paths
- Multi-step processes
- Content hierarchy display

## Best Practices

- Keep breadcrumb labels concise
- Ensure all items except last are clickable
- Test on mobile devices for usability
- Use consistent separator styling
