# DynTabs

Tab navigation with keyboard support and orientation variants.

## Overview

DynTabs organizes content into separate views accessible via tabs:
- Keyboard navigation (arrow keys)
- Horizontal and vertical orientations
- Controlled and uncontrolled modes
- Lazy loading support
- Icon and badge support

<Canvas of={Stories.Default} />

## Use Cases

- **Settings panels** - Organize settings into categories
- **Dashboards** - Switch between different data views
- **Multi-step forms** - Navigate form sections
- **Documentation** - Separate examples, API, code views

## Accessibility

### Keyboard Navigation

- **ArrowLeft/ArrowRight** - Navigate tabs (horizontal)
- **ArrowUp/ArrowDown** - Navigate tabs (vertical)
- **Home/End** - Jump to first/last tab
- **Tab** - Move focus to tab panel content

### Screen Reader Support

- Uses `role="tablist"`, `role="tab"`, `role="tabpanel"`
- `aria-selected` indicates active tab
- `aria-controls` links tabs to panels
- `aria-labelledby` associates panels with tabs

## Best Practices

✅ **Do:**
- Use clear, concise tab labels
- Limit number of tabs (typically 3-7)
- Show active tab clearly
- Load content lazily when appropriate
- Maintain state when switching tabs

❌ **Don't:**
- Use tabs for sequential steps (use stepper instead)
- Hide critical information in tabs
- Nest tabs within tabs
- Use too many tabs (>10)

## Examples

### Basic Tabs

```tsx
import { DynTabs } from '@dynui-max/core';

const items = [
  { value: 'overview', label: 'Overview', panel: <div>Overview content</div> },
  { value: 'details', label: 'Details', panel: <div>Details content</div> },
  { value: 'settings', label: 'Settings', panel: <div>Settings content</div> },
];

<DynTabs items={items} defaultValue="overview" />
```

### Controlled Tabs

```tsx
import { DynTabs } from '@dynui-max/core';
import { useState } from 'react';

function ControlledTabs() {
  const [activeTab, setActiveTab] = useState('tab1');

  const items = [
    { value: 'tab1', label: 'Tab 1', panel: <div>Content 1</div> },
    { value: 'tab2', label: 'Tab 2', panel: <div>Content 2</div> },
  ];

  return (
    <DynTabs 
      items={items} 
      value={activeTab} 
      onChange={setActiveTab}
    />
  );
}
```
