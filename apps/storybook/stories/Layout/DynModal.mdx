# DynModal

Accessible modal dialog component with focus management and portal rendering.

## Features

- **Focus Management**: Automatically traps focus within modal
- **Portal Rendering**: Renders outside normal DOM hierarchy
- **Keyboard Navigation**: ESC to close, proper tab ordering
- **Accessibility**: WCAG 2.1 AA compliant with ARIA attributes
- **Backdrop Control**: Optional backdrop click to close
- **Animation Support**: Smooth open/close transitions

## Usage Examples

- Confirmation dialogs
- Form modals
- Image galleries
- Settings panels

## Best Practices

- Always provide a clear close mechanism
- Use descriptive titles and content
- Test keyboard navigation thoroughly
- Consider mobile viewport constraints
