# DynIcon

Icon registry system with size and color token integration.

## Features

- **Icon Registry**: Centralized icon management system
- **Size Variants**: Multiple size options using design tokens
- **Color Integration**: Theme-aware color system
- **Accessibility**: Proper ARIA labels and semantic meaning
- **SVG Optimization**: Optimized SVG rendering
- **Custom Icons**: Support for custom icon registration

## Usage Examples

- Button icons
- Navigation indicators
- Status symbols
- Decorative elements

## Best Practices

- Use semantic icon names
- Provide alternative text for context
- Keep icon library lean and focused
- Ensure icons work at different sizes
