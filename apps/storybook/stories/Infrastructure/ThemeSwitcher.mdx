# ThemeSwitcher

Theme toggle component with button, toggle, and dropdown variants.

## Features

- **Multiple Variants**: Button, toggle switch, dropdown options
- **Theme Persistence**: Remembers user preference
- **Smooth Transitions**: Animated theme switching
- **Accessibility**: Keyboard navigation and screen reader support
- **System Preference**: Respects OS theme preference
- **Custom Themes**: Support for additional theme options

## Usage Examples

- Application headers
- Settings panels
- User preference controls
- Accessibility toolbars

## Best Practices

- Place in consistent, discoverable location
- Provide clear visual feedback
- Test with different themes
- Consider user preference persistence
