# DynTreeView

Hierarchical tree component with expand/collapse and selection functionality.

## Features

- **Hierarchical Data**: Support for nested tree structures
- **Expand/Collapse**: Interactive node expansion
- **Selection**: Single or multiple node selection
- **Keyboard Navigation**: Full keyboard support with arrow keys
- **Accessibility**: ARIA tree implementation
- **Custom Icons**: Configurable expand/collapse and node icons
- **Lazy Loading**: Support for dynamic data loading

## Usage Examples

- File system browsers
- Organizational charts
- Category hierarchies
- Decision trees

## Best Practices

- Limit initial expansion depth
- Provide clear visual hierarchy
- Use consistent iconography
- Implement efficient data loading for large trees
