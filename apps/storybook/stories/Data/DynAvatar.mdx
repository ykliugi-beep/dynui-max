# DynAvatar

User avatar component with image fallback to initials and multiple size variants.

## Features

- **Image Fallback**: Graceful fallback to initials when image fails
- **Size Variants**: Multiple size options (xs, sm, md, lg, xl)
- **Accessibility**: Proper alt text and ARIA attributes
- **Status Indicators**: Optional status dot for online/offline states
- **Responsive Images**: Optimized loading and display
- **Custom Styling**: Theme integration with design tokens

## Usage Examples

- User profiles
- Comment sections
- Team member lists
- Message threads

## Best Practices

- Always provide fallback text (name or initials)
- Use appropriate sizes for context
- Optimize images for performance
- Consider privacy implications
