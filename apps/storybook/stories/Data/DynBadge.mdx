# DynBadge

Status badge component with multiple variants and color options.

## Features

- **Color Variants**: Success, error, warning, info, neutral states
- **Size Options**: Small, medium, large sizing
- **Icon Support**: Optional leading or trailing icons
- **Dot Variant**: Minimal dot-only indicator
- **Accessibility**: Proper semantic meaning and contrast
- **Custom Content**: Support for numbers, text, or icons

## Usage Examples

- Status indicators
- Notification counts
- Labels and tags
- Progress states

## Best Practices

- Use semantic colors consistently
- Keep text content brief
- Ensure sufficient color contrast
- Consider international number formatting
