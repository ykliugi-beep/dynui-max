# DynTable

Advanced data table with sorting, selection, and loading states.

## Features

- **Sorting**: Click column headers to sort data
- **Row Selection**: Single or multiple row selection
- **Loading States**: Skeleton loading and empty states
- **Responsive Design**: Mobile-friendly table layout
- **Accessibility**: Screen reader support and keyboard navigation
- **Custom Renderers**: Custom cell rendering functions
- **Pagination**: Built-in pagination controls

## Usage Examples

- Data grids
- Admin dashboards
- Report tables
- Content management

## Best Practices

- Keep column count reasonable for readability
- Provide clear sorting indicators
- Use loading states for better UX
- Test with screen readers
